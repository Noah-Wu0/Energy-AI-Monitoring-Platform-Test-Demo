# Mock 数据策略

## 1. 结论

第一阶段采用：

```text
真实地理边界 + 公开产业背景 + 模拟监管数据
```

这条边界必须在产品、设计、代码和演示话术中保持一致。公开资料只用于建立可信背景和命名参考，不用于伪装真实监管数据。

## 2. 数据分层

| 层级 | 数据性质 | 可用范围 | 示例 | 页面标注 |
|---|---|---|---|---|
| L1 地理边界 | 真实公开数据 | 可直接用于地图底图和下钻 | Kazakhstan 全国边界、州级边界、曼吉斯套州边界 | “地图边界仅为示意，非官方测绘” |
| L2 产业背景 | 公开事实资料 | 可用于叙事、节点类型、命名灵感 | 阿克套港、曼吉斯套州油气产业、公开管道资料 | 标注来源，不做监管结论 |
| L3 监管业务数据 | 模拟数据 | 用于 demo 流程和交互 | 能耗曲线、预测带、异常事件、审计链、复核记录 | “数据为模拟演示” |
| L4 AI 输出 | 模拟/模板生成 | 用于解释产品能力 | AI 初判、建议核查、报告摘要 | “AI 输出为演示内容，需人工确认” |

## 3. 可直接使用的数据

### 3.1 地图边界

以下文件可直接进入前端地图开发：

- `assets/maps/kazakhstan-all-levels-topojson-v1.json`
- `assets/maps/kazakhstan-regions-v1.geojson`
- `assets/maps/mangystau-region-v1.geojson`
- `assets/maps/mangystau-districts-v1.geojson`

用途：

- 全国概览。
- 州级下钻。
- 曼吉斯套州特写。
- 阿克套市与州内区级边界展示。

### 3.2 公开产业线索

可作为 demo 背景：

- 阿克套港与油气运输关系。
- 曼吉斯套州油气产业地位。
- Uzen、Kalamkas、Karazhanbas、Dunga 等公开油田名称。
- Uzen-Zhetybay-Aktau、Kalamkas-Karazhanbas-Aktau 等公开管道线索。

使用边界：

- 可以作为节点命名灵感。
- 可以支撑“阿克套/曼吉斯套州是合理演示区域”的叙事。
- 不把公开产量、运量、储量直接混入监管事件判断。

## 4. 必须模拟的数据

以下数据第一阶段全部模拟：

- 企业实时能耗。
- 15 分钟级计量数据。
- AI 预测带。
- 异常事件。
- 设备状态。
- 计量站/泵站精确坐标。
- 管道实时流量和压力。
- 企业报送状态。
- 人工复核记录。
- 审计日志。
- 报告内容。

原因：

1. 公开渠道无法获取真实监管数据。
2. 真实计量点、泵站和设备坐标可能涉及安全敏感信息。
3. Demo 需要可控剧情，模拟数据更适合端到端演示。

## 5. 命名规则

模拟数据 ID 应明确表达 demo 属性。

推荐：

```text
demo-region-mangystau
demo-field-aktau-oil-zone
demo-metering-aktau-03
demo-event-night-load-001
demo-audit-001
```

避免：

```text
real-kmg-production
official-metering-station
actual-pipeline-pressure
```

企业或油田名称可使用“公开名称 + Demo 标识”的方式：

```text
阿克套油田能源作业区（Demo）
Uzen 示意油田节点
Aktau Port 示意节点
```

## 6. 页面标注规则

全局或关键页面需出现短标注：

```text
Demo 数据，非真实监管数据
```

地图页补充：

```text
地图边界仅作示意，非官方测绘
```

AI 面板补充：

```text
AI 初判仅供人工复核参考
```

报告页补充：

```text
本报告由模拟数据生成，用于演示流程
```

## 7. 自洽规则

模拟数据必须自洽：

- 异常事件时间必须落在时序数据范围内。
- 审计日志时间必须晚于异常检测时间。
- 人工复核状态必须能反映在事件卡片和报告中。
- 地图节点 ID 必须能关联到事件、证据和审计日志。
- 同一事件的严重程度颜色必须在地图、列表、报告中一致。

## 8. 禁止事项

- 不把模拟产量写成真实产量。
- 不用真实企业 logo 暗示授权。
- 不写“AI 已确认违规”。
- 不写“系统自动判定违法”。
- 不展示无法溯源的真实设备坐标。
- 不把新闻数据直接转成监管证据。
- 不把企业真实 logo 与模拟异常结论直接绑定。
- 不把国徽或政府机构名称用于暗示 demo 是正式上线系统。
