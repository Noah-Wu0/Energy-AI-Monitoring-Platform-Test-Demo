# 项目索引

## 项目定位

阿克套油田能源作业区 AI 监管闭环 Demo 是一个内部对齐和可点击演示项目。第一阶段用模拟数据展示“AI 监控、人在回路、全流程可审计”的监管闭环。

## 本地目录职责

| 目录 | 用途 |
|---|---|
| `docs/00-project/` | 项目总览、文件规范、目录规范、协作约定 |
| `docs/01-product/` | PRD、用户流程、演示脚本、需求变更记录 |
| `docs/02-design/` | 设计原则、页面结构、视觉规范、交互规范 |
| `docs/03-engineering/` | 技术架构、工程约定、接口说明、实现计划 |
| `docs/04-data/` | Mock 数据契约、字段字典、样例数据说明 |
| `docs/05-assets/` | 素材归档规则、地图/图标/截图使用说明 |
| `docs/06-handoff/` | 多模型协作任务说明、交接记录、检查清单 |
| `docs/research/` | 外部案例调研、参考链接、设计启发 |
| `assets/` | 本地素材工作区 |
