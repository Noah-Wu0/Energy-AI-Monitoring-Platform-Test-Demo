# 品牌与机构标识使用规范

## 1. 适用范围

本规范覆盖 demo 中可能出现的政府机构标识、国徽、企业 logo、企业名称和临时项目标识。

核心原则：

1. 内部 demo 可以使用公开标识做语境参考。
2. 不暗示官方授权、合作或背书。
3. 不把真实 logo 与模拟监管数据绑定成真实结论。
4. 国家象征和企业商标都要谨慎使用。

## 3. 能源部标识策略

目前未找到能源部独立 logo。哈萨克斯坦政府机构页面通常使用国家机构平台和国徽体系。第一阶段 demo 采用：

```text
国徽 + 哈萨克斯坦共和国能源部
```

推荐中英文/哈文名称：

```text
哈萨克斯坦共和国能源部
Ministry of Energy of the Republic of Kazakhstan
Қазақстан Республикасы Энергетика министрлігі
```

使用要求：

- 只在 header、封面、报告页页眉等机构上下文中使用。
- 不做颜色重绘、形状改造、裁切变形。
- 可缩小、可降低透明度做背景水印，但不做装饰化再设计。
- 页面需保留 demo 说明，避免被误认为正式系统。

## 4. 企业 logo 策略

KMG logo 可用于内部 demo 的企业节点和企业详情面板，但必须遵守：

- 不暗示 KMG 或其子公司授权。
- 不与“违规”“违法”“真实异常”等确定性监管结论直接绑定。
- 不用于商业宣传物料。
- 如对外演示，应优先改为文字标识或抽象企业图标，除非获得授权。

对于 KMG 子公司：

- 第一阶段优先使用 KMG 伞形标识 + 子公司名称。
- 如需区分管道、港口、油田等业务主体，优先用行业图标和文字，不急于下载所有企业 logo。

## 5. 页面免责声明

推荐放在 footer、关于弹窗或报告页末尾：

```text
本演示中的企业名称、机构名称和标识仅作行业背景参考，不暗示与相关机构或企业存在合作、授权或背书关系。所有监管数据、异常事件和 AI 输出均为模拟生成。
```

短版：

```text
Demo 数据与标识仅作演示参考，不代表真实监管结论或官方授权。
```

## 6. 禁止事项

- 不修改国徽构图。
- 不把国徽做成装饰性图案或大面积背景花纹。
- 不把企业 logo 用在风险结论标题旁边。
- 不把模拟异常写成企业真实异常。
- 不把第三方 logo 库下载的标识视为授权资产。
- 不在外部宣传稿、商业材料中使用这些标识，除非另行确认授权。

## 7. 后续缺口

- 如需对外正式演示，需要确认国徽和政府机构名称的使用边界。
- 如需大量使用企业 logo，需要向企业官网或品牌中心确认授权。
- 可设计一个项目专属临时标识，用于内部开发环境和非正式页面，减少真实标识使用频率。
- 对能源部下属机构，优先采用“名称 + 统一线性图标 + 业务线颜色”，logo 只放在机构详情层，避免首页像资料拼贴。
- 历史飞书客户资料中的图片已归档为参考素材，正式页面使用前需要二次核验来源。
