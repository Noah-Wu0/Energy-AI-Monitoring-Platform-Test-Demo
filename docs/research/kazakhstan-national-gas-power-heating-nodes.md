## 五、跨领域联动观察

各能源领域之间存在关键的交叉依赖关系，对 Demo 场景设计至关重要：

| 联动类型 | 具体依赖 | Demo 场景示例 |
|---|---|---|
| **气→电** | 燃气电厂（EG-01 Atyrau 300MW 等）依赖天然气供应 | 管道减供 → 发电出力下降 |
| **煤→电** | Ekibastuz 全国最大煤电基地依赖 Bogatyr 煤矿 | 煤矿事故 → 全哈 42% 电力风险 |
| **电→油** | 油田泵站/计量站依赖电网供电 | 核电站断电 → Mangystau 多油田停产（Interfax 报道的真实事件） |
| **电→热** | CHP 热电联产同时供电供热 | 机组故障 → 停电 + 停暖双重打击 |
| **气→热** | Turan GTS / 城市燃气锅炉依赖管道气 | 气源中断 → 首都 Astana 大面积停暖 |
| **油→气** | 伴生气处理依赖油田生产稳定性 | 油田停产 → 伴生气减供 → GPP 停工 |
